# 头像图片命名规范

## 说明
此目录用于存放应用的头像图片资源。

## 命名规则
- **文件名格式**: `avatar_[描述].png` 或 `avatar_[描述].jpg`
- **描述建议**:
  - `default`: 默认头像
  - `user`: 用户头像
  - `admin`: 管理员头像
  - 其他描述性名称

## 示例
- `avatar_default.png` - 默认头像
- `avatar_user_male.png` - 男性用户头像
- `avatar_admin.png` - 管理员头像

## 注意事项
- 图片会被自动压缩和调整大小
- 建议使用正方形图片以获得最佳显示效果
- 支持 PNG 和 JPG 格式
