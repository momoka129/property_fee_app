## 📱 功能展示路径

### 主要流程
1. **启动应用** → 进入主页仪表盘
2. **查看账单** → 点击"Bills"图标 → 查看账单列表 → 点击账单详情 → 模拟支付
3. **提交报修** → 点击"Repairs"图标 → 点击"New Request"按钮 → 填写表单
4. **查看公告** → 点击"Announcements"或主页公告卡片 → 查看详情
6. **取包裹** → 点击"Packages"图标 → 查看待取包裹 → 标记已取
7. **预订设施** → 点击"Amenities"图标 → 选择设施 → 预订
8. **停车管理** → 点击"Parking"图标 → 查看停车位 → 生成二维码
